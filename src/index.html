01. Google Calculator Clone

In this project, I built a functional replica of Google's calculator using HTML, CSS, and JavaScript. The calculator has the same layout and design as Google's calculator, and I added all of the basic arithmetic operations, including the ability to handle decimal and negative numbers.

The calculator also has extra features such as a clear button that clears the input field and the history, a backspace button to delete the last digit entered, and an equal button that evaluates the expression and displays the result. Additionally, I added a history section that displays the expression and result of the previous calculation.

Building a replica of Google's calculator was a challenging but rewarding experience. I had to research and implement the correct order of operations and handling input of decimal and negative numbers. This project helped me improve my HTML, CSS, and JavaScript skills and gave me a better understanding of how calculators work. Overall, I'm proud of the functional replica I built and excited to add it to my portfolio as a demonstration of my web development skills.

02. Jovian Careers Website

Jovian careers website is a project built using modern web technologies like Express JS, Bootstrap, and HTML. The website is a clone of the original Jovian careers site, with the aim of providing a platform that replicates the seamless job application process provided by Jovian, so that job seekers can experience the same ease of use when browsing engineering job listings.

The website features a clean and user-friendly interface, making job hunting an enjoyable experience for anyone interested in software engineering roles. Job seekers can browse through various job listings, filter them based on their interests, and apply to them using a simple application form.
